2. The LVHA order 📚

When you style links, the order of the pseudo‑classes matters. The classic rule is: a:link → a:visited → a:hover → a:active

If you put them in a different order, some rules can “override” others in ways that feel broken.

Correct order
a:link { /* default link style */ }
a:visited { /* after you clicked it before */ }
a:hover { /* when the mouse is over it */ }
a:active { /* while you are clicking it */ }

Key idea
.combo a { position: relative; text-decoration: none; }
.combo a::after {
  content: "";
  position: absolute;
  left: 0; bottom: -2px;
  width: 0%; height: 2px;
  background: linear-gradient(90deg, #f97316, #ec4899, #6366f1);
  transition: width 0.2s ease;
}

.combo a:hover::after { width: 100%; }
