3. :visited (simulated)

Click Me

CSS

.visited-demo:visited {
    color: purple;
}

Browsers limit :visited behavior for privacy, so this demo simulates it.

4. :hover

Hover Over Me

CSS

.hover-demo:hover {
    color:red;
    text-decoration:underline;
}

Appears when your mouse is over the link.

5. :active

Click And Hold

CSS

.active-demo:active {
    color:green;
}

Visible while the mouse button is held down.

6. Complete LVHA Example

Combined LVHA Link
.lvha-demo:link { color: blue; }
.lvha-demo:visited { color: purple; }
.lvha-demo:hover { color: red; }
.lvha-demo:active { color: green; }

The recommended order is Link → Visited → Hover → Active.

7. Wrong Order Example

Wrong Order Link
.bad-order:hover { color:red; }
.bad-order:link { color:blue; }
.bad-order:visited { color:purple; }
.bad-order:active { color:green; }

Pseudo-class order matters. LVHA helps avoid unexpected results.
